## How It Works

Image Forge leverages cutting-edge web technologies to provide a seamless image conversion experience:

1. **File Selection**: Drag & drop or select images using the file picker.
2. **Preview**: View your selected images in a sleek carousel.
3. **Conversion**: Choose your desired output format and start the conversion process.
4. **Processing**: Images are converted client-side using HTML5 Canvas and Web Workers.
5. **Compression**: Advanced algorithms optimize file sizes without significant quality loss.
6. **Download**: Converted images are packaged into a zip file for easy downloading.

## Getting Started

To run Image Forge locally, follow these steps:

```bash
# Clone the repository
git clone https://github.com/mohdyahyamahmodi/image-forge.git

# Navigate to the project directory
cd image-forge

# Open index.html in your preferred browser
open index.html
```

No server setup required! Image Forge runs entirely in your browser.

## Usage

1. **Upload Images**: 
   - Drag and drop images onto the designated area, or
   - Click "Select Files" to use the file picker

2. **Choose Format**: 
   Select your desired output format from the dropdown menu

3. **Convert**: 
   Click the "Convert Images" button to start processing

4. **Download**: 
   Once conversion is complete, download your optimized images as a zip file

## FAQ

<details>
<summary><strong>Is my data safe?</strong></summary>
Yes! All processing happens in your browser. Your images never leave your device.
</details>

<details>
<summary><strong>What's the maximum file size?</strong></summary>
Image Forge can handle files up to 100MB each, with a total limit of 500MB per session.
</details>

<details>
<summary><strong>Can I convert multiple formats at once?</strong></summary>
Currently, all images are converted to a single chosen format per session. Multiple format support is planned for future updates.
</details>
